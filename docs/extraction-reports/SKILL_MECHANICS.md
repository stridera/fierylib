# Skill Mechanics Reference

Complete reference for all skills in FieryMUD.

**Total Skills**: 129

## Skill Categories

### Combat Skills
Active combat abilities that require player input.

### Passive Skills
Always-on bonuses and proficiencies.

### Utility Skills
Non-combat abilities for exploration and interaction.

### Racial Skills
Race-specific innate abilities.

## Combat Skills


### Two-Back
- **ID**: `SKILL_2BACK`
- **Description**: Dual backstab

### Backstab
- **ID**: `SKILL_BACKSTAB`
- **Description**: Attack from behind for massive damage
- **Damage**: 2x-5x (skill 0-200: 2x, 201-400: 3x, 401-600: 4x, 601+: 5x)
- **Requirements**: piercing weapon, target not fighting, hidden or behind
